This is the Map Editor : You can add a desciption of your map here. The map editor is useful for helping you get a general idea of where the players are located.

Button Explaination :

: places you into viewing mode allowing you to check things you have placed.
: opens your cabinet of tools and map components once a tool or component is selected the icon will replace this one. Click on that icon to reopen the cabinet.
: opens a mail client so you can send messages to your players.
: is used to add NPC which you have created to the map.
: is used to add encounters which you have created to the map.
: is used to create a new zone and connect it to this map.
: is used to add items to the map.
: is used to add additional notes to the map you can also view these notes via the side interface.
